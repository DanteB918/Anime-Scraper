Anime Scraper API Documentation

A JavaScript-based API for scraping anime data from Anitaku/GoGoAnime

Recent Episodes

/index.html?page=2

Returns a JSON list of recent anime episodes from the Anitaku/GoGoAnime homepage.

Parameters

Parameter Type Required Description
page integer No Page number for pagination (defaults to 1)

Example Request

GET /index.html?page=2

Example Response

{ "Naruto Shippuden Episode 500": { "episode": "Episode 500", "img": "https://gogocdn.net/images/anime/N/naruto-shippuden.jpg", "url": "https://anitaku.bz/naruto-shippuden-episode-500" }, "One Piece Episode 1080": { "episode": "Episode 1080", "img": "https://gogocdn.net/images/anime/O/one-piece.jpg", "url": "https://anitaku.bz/one-piece-episode-1080" }, // More episodes... }

Episode Details

/episode.html?show=one-piece&episode=1080

Returns detailed information about a specific anime episode, including video sources and navigation links.

Parameters

Parameter Type Required Description
show string Yes The show name in URL-friendly format (e.g., "one-piece", "naruto-shippuden")
episode integer Yes The episode number

Example Request

GET /episode.html?show=one-piece&episode=1080

Search Results

/search.html?search=demon+slayer&page=1

Returns search results for anime based on the provided query.

Parameters

Parameter Type Required Description
search string Yes The search query (e.g., "demon slayer", "my hero academia")
page integer No Page number for pagination (defaults to 1)

Example Request

GET /search.html?search=demon+slayer&page=1

Example Response

{ "Demon Slayer": { "release": "2019", "img": "https://gogocdn.net/cover/kimetsu-no-yaiba.png", "url": "https://anitaku.bz/category/kimetsu-no-yaiba" }, "Demon Slayer: Kimetsu no Yaiba Swordsmith Village Arc": { "release": "2023", "img": "https://gogocdn.net/cover/kimetsu-no-yaiba-katanakaji-no-sato-hen.png", "url": "https://anitaku.bz/category/kimetsu-no-yaiba-katanakaji-no-sato-hen" }, "Demon Slayer Movie: Mugen Train": { "release": "2020", "img": "https://gogocdn.net/cover/kimetsu-no-yaiba-movie-mugen-ressha-hen.png", "url": "https://anitaku.bz/category/kimetsu-no-yaiba-movie-mugen-ressha-hen" }, // More search results... }

Important Notes

  • All endpoints return data in JSON format
  • Requests may be subject to CORS restrictions when used in browser environments
  • The API uses a CORS proxy (allorigins.win) to fetch data from Anitaku/GoGoAnime
  • Response data structure may change if the source website's HTML structure changes
